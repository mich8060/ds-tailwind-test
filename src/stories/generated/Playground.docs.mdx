<Markdown>{`# Playground

Interactive component explorer with live canvas, inspector panel, and code generation.

## When to Use
- Building demo/documentation pages for components
- Interactive "try it out" experiences for component props
- Generating code snippets for component configurations

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| \`component\` | \`React.Component\` | — | The component to render in the canvas |
| \`propDefinitions\` | \`object\` | \`{}\` | Prop definitions for inspector controls (see below) |
| \`title\` | \`string\` | \`"Playground"\` | Playground title |
| \`initialProps\` | \`object\` | \`{}\` | Initial prop values |
| \`renderComponent\` | \`function\` | — | Custom render function \`(props) => ReactNode\` |

### propDefinitions Format

Each key maps to a control definition:

\`\`\`jsx
{
  propName: {
    type: "select" | "text" | "boolean" | "number" | "icon",
    label: "Display Label",
    default: defaultValue,
    options: [{ value: "val", label: "Label" }],  // for select
    min: 0, max: 100, step: 1,                    // for number
    placeholder: "Enter text..."                    // for text
  }
}
\`\`\`

## Examples

### Button playground
\`\`\`jsx
<Playground
  component={Button}
  propDefinitions={{
    label: { type: "text", label: "Label", default: "Click Me" },
    appearance: {
      type: "select",
      label: "Appearance",
      default: "primary",
      options: [
        { value: "primary", label: "Primary" },
        { value: "soft", label: "Soft" },
        { value: "outline", label: "Outline" },
        { value: "ghost", label: "Ghost" },
      ],
    },
    size: {
      type: "select",
      label: "Size",
      default: "default",
      options: [
        { value: "large", label: "Large" },
        { value: "default", label: "Default" },
        { value: "small", label: "Small" },
      ],
    },
    icon: { type: "icon", label: "Icon", default: "" },
    disabled: { type: "boolean", label: "Disabled", default: false },
  }}
/>
\`\`\`

### With custom render function
\`\`\`jsx
<Playground
  component={Card}
  propDefinitions={cardProps}
  renderComponent={(props) => (
    <div style={{ width: 300 }}>
      <Card {...props}>
        <p>Custom card content</p>
      </Card>
    </div>
  )}
/>
\`\`\`

## Features
- **Canvas**: Live component preview with current props
- **Inspector**: Property controls with select, text, number, boolean, and icon types
- **Code panel**: Auto-generated import + usage code with syntax highlighting
- **Copy button**: One-click code copy to clipboard
- Icon props auto-detect and show icon picker from Phosphor icon library

## Note
- Not typically used in production apps — this is for documentation/demo pages
- The \`component.name\` is used for code generation (ensure components have display names)`}</Markdown>
