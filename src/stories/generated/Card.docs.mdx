<Markdown>{`# Card

Navigation card component for linking to pages with icon, title, and description.

## When to Use
- Overview/index pages with links to sub-pages
- Dashboard navigation tiles
- Feature showcases

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| \`to\` | \`string\` | — | Route path to navigate to (uses React Router \`Link\`) |
| \`title\` | \`string\` | — | Card title text |
| \`description\` | \`string\` | — | Card description text |
| \`icon\` | \`ReactNode\` | — | SVG icon or React element to display |
| \`className\` | \`string\` | \`""\` | Additional CSS classes |

## Examples

\`\`\`jsx
<Card
  to="/users"
  title="Users"
  description="Manage user accounts and permissions"
  icon={<Icon name="Users" size={24} />}
/>
\`\`\`

### Grid of cards
\`\`\`jsx
<Flex gap="16" wrap>
  <Card to="/dashboard" title="Dashboard" description="Overview" icon={<Icon name="ChartBar" size={24} />} />
  <Card to="/settings" title="Settings" description="Configure app" icon={<Icon name="Gear" size={24} />} />
</Flex>
\`\`\`

## Note
- Renders as a React Router \`<Link>\` — must be used within a \`<Router>\` context
- The entire card is clickable`}</Markdown>
