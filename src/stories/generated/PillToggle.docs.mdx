<Markdown>{`# PillToggle

Segmented control with pill-shaped toggle buttons for switching between views or modes.

## When to Use
- View mode switching (Grid vs. List, Map vs. Satellite)
- Small option groups (2–4 options) as an alternative to tabs
- Compact controls for toolbars

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| \`options\` | \`array\` | \`[]\` | Options: \`[{ value, label, icon? }]\` |
| \`value\` | \`string\` | — | Currently selected value |
| \`onChange\` | \`function\` | — | Callback \`(value: string) => void\` |
| \`size\` | \`string\` | \`"default"\` | \`"small"\`, \`"default"\` |
| \`className\` | \`string\` | \`""\` | Additional CSS classes |

## Examples

\`\`\`jsx
<PillToggle
  options={[
    { value: "grid", label: "Grid", icon: "GridFour" },
    { value: "list", label: "List", icon: "List" },
  ]}
  value={viewMode}
  onChange={setViewMode}
/>
\`\`\`

### Text-only
\`\`\`jsx
<PillToggle
  options={[
    { value: "day", label: "Day" },
    { value: "week", label: "Week" },
    { value: "month", label: "Month" },
  ]}
  value={calendarView}
  onChange={setCalendarView}
/>
\`\`\``}</Markdown>
