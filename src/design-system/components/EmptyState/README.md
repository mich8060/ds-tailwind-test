# EmptyState

Tier: 2

Purpose:
- Provide a clear fallback UI when data is empty.

## Props

- `title` (required): Primary heading content.
- `description` (optional): Supporting copy.
- `icon` (optional): Phosphor icon name. Defaults to `"Inbox"`.
- `iconSize` (optional): Icon size in px. Defaults to `32`.
- `action` (optional): Primary action node (typically a `Button`).
- `secondaryAction` (optional): Secondary action node.
- `align` (optional): `"center"` or `"left"`. Defaults to `"center"`.
