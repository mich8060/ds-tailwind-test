# Tabs

## Key Props

- `appearance`: `"underline" | "block" | "block-inverted"`
- `orientation`: `"horizontal" | "vertical"` (default: `"horizontal"`)

## Structure Compliance

- Uses semantic token styles from `var(--uds-...)`.
- BEM naming with `.uds-tabs` as the block prefix.
- Variants and defaults are defined in `Tabs.spec.ts`.
- Public exports are limited to component and types through `index.ts`.
