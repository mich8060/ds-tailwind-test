# Text

Tier: 2

## Props
- `as` (semantic tag): `"p"` by default
- `variant`: display/heading/body variants
- `weight`: regular | medium | semibold | bold
- `leading`: tight | regular | loose

## Notes
- Responsive behavior is achieved by overriding primitive tokens at breakpoints.
- Includes legacy class aliases (e.g. `.uds-display-96-bold`) for migration.
